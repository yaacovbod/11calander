# פרומפט לקלוד קוד — דשבורד סיכויי בגרות שכבת יא׳

## הקשר

אני רכז שכבת יא׳ בבית הספר נעימת הלב בחריש.  
אני רוצה לבנות אפליקציית web שמחשבת ומציגה את סיכויי הבגרות של כל תלמיד בשכבה.  
השכבה כוללת 84 תלמידים בכיתות יא1–יא6.

הנתונים מגיעים מקובץ Excel שאני מתחזק (מצורף: `סדין_בגרויות_נעימת_הלב_מעודכן.xlsx`).  
הוא מכיל 5 גיליונות: `הגדרות`, `תלמידים`, `ציוני_בגרות`, `ציוני_תעודה`, `דוח`.

### רכיב 1: ציוני בגרויות שנעשו (s1 — משקל 55%)

זהו הרכיב הקריטי. הוא עצמו ממוצע משוקלל של הסתברויות לעבור כל מקצוע:

#### **כלל 50% השכבה**

לפני חישוב כל מקצוע בודקים: האם 50%+ מהשכבה ניגשו לבחינה הזאת?

```typescript
function isCohortDone(field: keyof BagrutScores, allScores: BagrutScores[]): boolean {
  const n = allScores.length;
  const count = allScores.filter(s => s[field] !== null && s[field] !== 0).length;
  return count / n >= 0.50;
}
```

- אם **כן**: חסר = תלמיד לא ניגש = **0**
- אם **לא**: חסר = שכבה עדיין לא עשתה = **ניטרלי (null, לא נכנס לחישוב)**

**חריגים לכלל:**
- `math_35173` (שאלון ראשון 3 יח"ל) — לא חל הכלל, רק תלמידי 3 יח"ל עושים אותו
- מגמות (ביולוגיה, מוט"ל, שפות) — ניטרלי אם חסר, **0 רק אם כתוב 0 מפורשות**
- **היסטוריה** — תמיד ניטרלי אם חסר (המורה מוסיף 0 ידנית לפי שיקול דעת)

#### **לשון (11281)**

מבנה: `exam * 0.70 + school * 0.30`  
הערכה פנימית מובטחת: **85**  
לכן: `final ≈ exam * 0.70 + 85 * 0.30 = exam * 0.70 + 25.5`

```typescript
function probLashon(
  exam: number | null,
  isSpecialEd: boolean,
  isLateJoiner: boolean,
  cohortDone: boolean,
  settings: Settings
): number | null {
  if (!exam || exam === 0) {
    if (isLateJoiner) return null;          // הצטרף מאוחר — ניטרלי
    if (isSpecialEd) return settings.specialEdLashonNoExam;  // 80
    return cohortDone ? 0 : null;           // לא הוגש / שכבה לא עשתה
  }
  const final = exam * 0.70 + settings.guaranteedInternalGrade * 0.30;
  if (final >= 60) return 100;
  if (final >= 55) return 95;
  if (final >= 52) return 70;
  if (final >= 48) return 40;
  if (final >= 44) return 15;
  return 5;
}
```

**שם לב:** תלמידות שהצטרפו מאוחר (מנשוב אלינה, סויסה אלין) — `isLateJoiner = true`.

#### **מקצועות הרוח (תנ"ך, אזרחות, ספרות)**

מבנה: `online * 0.35 + exam * 0.35 + school * 0.30`  
ערכים מובטחים: online ≥ 70, school ≥ 85  
ציון מובטח לפני בחינה: `70 * 0.35 + 85 * 0.30 = 49.5 ≈ 50`  
לעבור 55: צריך exam ≥ 15 בלבד

```typescript
function probSpirit(
  exam: number | null,
  isSpecialEd: boolean,
  cohortDone: boolean,
  settings: Settings
): number | null {
  if (!exam || exam === 0) {
    if (isSpecialEd) return null;  // חינ"מ — תוכנית מיוחדת — ניטרלי
    return cohortDone ? 0 : null;
  }
  const guaranteed = settings.guaranteedOnlineTasksGrade * 0.35
                   + settings.guaranteedInternalGrade * 0.30;
  const final = exam * 0.35 + guaranteed;
  if (final >= 60) return 100;
  if (final >= 55) return 98;
  if (final >= 52) return 80;
  if (final >= 50) return 60;
  return 30;
}
```

#### **היסטוריה**

זהה ל-`probSpirit` אבל: חסר = **תמיד ניטרלי (null)** — ללא קשר לכלל 50%.  
המורה מוסיף 0 ידנית בגיליון Excel.

#### **חישוב s1 הסופי**

הציון מבוסס על **הוליסטיקה של כל המקצועות**:  
הציון הנמוך ביותר שולט (כי בגרות = כולם חייבים לעבור).

```typescript
function calcBagrutScore(probs: Record<string, number>): number {
  const values = Object.values(probs);
  if (values.length === 0) return 40;
  const min = Math.min(...values);
  const avg = values.reduce((a, b) => a + b, 0) / values.length;
  return min * 0.55 + avg * 0.45;
}
```

function calcSchoolGradesScore(grades: SchoolGrades): number {
  const core = [grades.civics, grades.english, grades.history,
                grades.hebrew, grades.math, grades.bible];
  const valid = core.filter(g => g !== null) as number[];
  if (valid.length === 0) return 50;
  const avg = valid.reduce((a, b) => a + b, 0) / valid.length;
  if (avg >= 75) return 90;
  if (avg >= 65) return 70;
  if (avg >= 55) return 50;
  return 25;
}

function calcMathEnglishLevel(student: Student, bagrut: BagrutScores): number {

## קובץ Excel המצורף — מבנה הגיליונות

### `הגדרות`
כל המשקלים, הספים והכללים — תאים צהובים = ניתנים לשינוי.

### `תלמידים`
עמודות: ת"ז, שם מלא, כיתה, חינ"מ (כן/לא), הצטרף מאוחר (כן/לא), אחוז היעדרות, נוכחות (נוסחה), יח"ל מתמטיקה, יח"ל אנגלית, הערות.

### `ציוני_בגרות`
- ת"ז כמפתח ראשי
- עמודות קלט (צהוב): שאלוני מתמטיקה, ציוני בחינה לכל מקצוע, מרכיבי אנגלית, מגמות
- עמודות חישוב (כחול): ציונים סופיים (נוסחאות)
- **אל תמחק נוסחאות בעמודות הכחולות**

### `ציוני_תעודה`
ציון מחצית בכל מקצוע. ממוצע ליבה — נוסחה.

### `דוח`
ממוין מהנמוך לגבוה. מעודכן בכל הרצת חישוב. לא לערוך ידנית.
